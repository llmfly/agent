# 汽车贸易分析逻辑参考

本文件提供针对汽车行业（特别是新能源汽车出海）的贸易数据解读逻辑，旨在提升研报的专业深度。

## 1. 产品结构解读 (Product Mix)
- **NEV 转型信号**：对比 8703 (乘用车) 下的不同子类。若 870322 (混动) 和 870380 (纯电) 的份额持续上升且超过传统燃油车，判定为“成功实现新能源转型”。
- **产业链输出**：关注 8708 (零部件) 与 8507 (动力电池) 的同步增长。若整车出口伴随大量电池和底盘件输出，说明企业在当地建立了较为完整的供应链体系或采用了高度集成化出口模式。
- **价值链位置**：分析 `amount / quantity` (单车均价)。高均价市场 $\rightarrow$ 品牌溢价/高端车型渗透；低均价市场 $\rightarrow$ 规模抢占/入门级车型主导。

## 2. 市场表现解读 (Market Performance)
- **增长极识别**：结合增速与规模。
    - 高规模 + 高增速 $\rightarrow$ 战略核心市场 (Core Engine)。
    - 低规模 + 高增速 $\rightarrow$ 潜力爆发市场 (Emerging Star)。
- **风险预警**：若某个核心市场的 `customer_count` 为 1 且金额巨大，判定为“单点依赖风险极高”，建议在报告中明确提出“渠道去中心化”建议。

## 3. 竞争格局解读 (Competition)
- **锚定市场 (Anchor Market)**：在汽车行业，通常以一个核心 HS 编码 + 一个核心国家作为锚定市场。
- **份额解读**：若份额接近 100%，需审视是否因 HS 编码定义过细。在汽车领域，应尝试扩大 HS 编码范围（如从 8703.22 扩大到 8703）来验证真实的竞争压力。

## 4. 战略建议逻辑
- **准入路径**：验证 (小批量/特定型号) $\rightarrow$ 试单 (核心经销商) $\rightarrow$ 放量 (全渠道/本地化组装)。
- **配套建议**：针对高增长市场，建议由“整车出口”升级为“零件配套 + 本地组装 (CKD/SKD)”，以降低关税压力并提升本地化率。
