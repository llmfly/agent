二、研究背景与动机

2.1 问题定义

本文聚焦于老年人活动识别（Elderly Activity Recognition）任务。随着人口老龄化加剧，空巢老人比例不断上升，利用计算机视觉技术对老年人日常活动进行智能监测具有重要意义。

2.2 核心挑战

与常规动作识别相比，老年人活动识别面临以下独特挑战：

  • 个体动作与人-物交互并存：老年人活动中既包含纯身体动作（个体动作），也包含与物体交互的行为（如吹头发、梳头）。
  • 局部交互微小：许多人-物交互动作是局部性的（如手部细微动作），容易被忽略。
  • 动作幅度不明显：老年人动作幅度较小，特征不显著。
  • 相似动作混淆：某些动作在时空轨迹上高度相似，仅靠微小局部差异才能区分。

典型案例：
「吹头发」vs「梳头」—— 动作轨迹和幅度高度相似，仅交互物体不同（吹风机 vs 梳子）
「打电话」vs「玩手机」—— 大部分运动和交互对象相同，仅局部手部动作有差异

三、核心方法：ESE-FN

3.1 总体框架

ESE-FN（Expansion-Squeeze-Excitation Fusion Network）的整体框架包含四个核心模块：

  1. 特征提取模块：RGB → ResNeXt101；骨架 → Shift-GCN
  2. 模态融合模块（M-Net）：通过模态级ESE注意力实现模态间粗粒度融合
  3. 通道融合模块（C-Net）：通过通道级ESE注意力实现通道间细粒度融合
  4. 多模态损失（Multi-modal Loss, ML）：保持单模态与融合特征的一致性
RGB视频 → ResNeXt101 → RGB特征 fr 骨架序列 → Shift-GCN → 骨架特征 fs ↓ 特征拼接 → M-Net（模态级ESE融合） → C-Net（通道级ESE融合）→ 融合特征 frs ↓ 多模态损失 L = α×L_rs + β×(min(L_r,L_s)-L_rs)

3.2 SENet回顾与改进

SENet通过Squeeze（全局平均池化）和Excitation（FC+Sigmoid）两步学习通道级注意力。本文的创新在于引入Expansion（扩展）步骤，形成ESE结构，实现从局部和全局两个视角特征交互。

3.3 M-Net（模态融合网络）

M-Net通过模态级ESE注意力（M-ESEA）实现模态间粗粒度融合的三个步骤：

  1. 模态级扩展：多个不同卷积核的卷积层堆叠，从局部视角扩展模态信息
  2. 模态级压缩：全局平均池化，从全局视角获得模态级表示
  3. 模态级激励：FC层+Sigmoid学习M-ESEA注意力权重，更新特征

3.4 C-Net（通道融合网络）

C-Net通过通道级ESE注意力（C-ESEA）实现通道间细粒度融合的三个步骤：

  1. 通道级扩展：单层卷积扩展通道信息（轻量化设计）
  2. 通道级压缩：全局平均池化获得通道级表示
  3. 通道级激励：FC层+Sigmoid学习C-ESEA注意力权重

设计思路：M-Net（粗融合，多层卷积）→ C-Net（细融合，单层卷积），逐步递进。

3.5 多模态损失（ML）

公式：L = α × L_rs + β × (min(L_r, L_s) − L_rs)

创新点：引入惩罚项，促使融合特征的预测损失不低于最优单模态。超参数最优值 α=0.7, β=0.3。

五、实验结果

5.1 消融实验

M-Net、C-Net、ML三个组件均正向贡献，完整ESE-FN最优。ESE-FN比SENet高出0.6%，验证Expansion步骤有效性。双重融合策略（模态+通道）优于单一融合策略。

5.2 老年人活动识别（ETRI-Activity3D）

ESE-FN准确率：95.9% ✦ SOTA
比前SOTA方法FSA-CNN（93.7%）高出 2.2%

六、论文贡献总结

  1. 提出ESE-FN框架：首次将Expansion引入SENet家族，形成ESE注意力结构用于多模态特征融合
  2. 设计M-Net和C-Net：分别从模态级和通道级捕获特征依赖，实现从粗到细的递进式融合
  3. 提出多模态损失ML：通过惩罚项保持单模态与融合特征之间的一致性
  4. 取得SOTA性能：ETRI-Activity3D上95.9%，NTU RGB+D上验证泛化能力

八、讨论与展望

本文在SENet基础上首次引入Expansion步骤，实现上采样和下采样的特征交互，是SENet家族中的新尝试。ESE-FN不仅限于老年人活动识别，还可推广到一般特征学习和多模态特征融合任务，为后续研究提供了新的思路。
